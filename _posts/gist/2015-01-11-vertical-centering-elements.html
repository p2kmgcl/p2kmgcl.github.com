Know a different way? Please tell me or fork this experiment

  • fw is not necesary in "production"
  • f represents the body element
  • wrapper can contain anything (including text)
  • Width and height must be defined (can be %)
  • CSS Transforms should work
  • Antialiasing could appear
  • Needs a third wrapping element
  • Wrapper fills everything
  • Content must be inlined
  • Height can't be a % value
  • Line-height must be specified
