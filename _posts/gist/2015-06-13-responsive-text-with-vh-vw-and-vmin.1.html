Feeling Responsive

Resize me vertically and/or horizontally

- - - /* I've just learnt this here: http://tutorialzine.com/2015/05/simplify-your-stylesheets-with-the-magical-css-viewport-units/ vw means 100% with of the viewport vh means 100% height of the viewport vmin means min(vw, vh) Oh, and there's full support from ie9: http://caniuse.com/#feat=viewport-units :| */ .sandboxWrapper { background-color: #CBE86B; color: #1C140D; font-family: 'Suranna'; text-align: center; } h1, h2 { margin: 0; line-height: 1.3em; font-weight: 400; } h1 { margin-top: 10%; font-size: 9vmin; } h2 { font-size: 4vmin; } - - -
